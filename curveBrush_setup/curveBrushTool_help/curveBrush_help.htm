curveBrushTool 

Ø   1、OverView：

curveBrushTool is a tool which is used to brush the shape of a nurbscurve in Maya.

Maya Version 8.5.

Ø   2、Setup：

Copy  “ curveBrushProperties.mel，curveBrushRunTimeCommands.mel，curveBrushValues.mel，curveBrushToolIni.mel，

curveBrushTool_help ” to My Documents\maya\8.5\scripts；

Copy “shelf_YinHeTS.mel” to My Documents\maya\8.5\prefs\shelves；

Copy “curveBrushTool.xpm” to C:\Program Files\Autodesk\Maya8.5\icons；

Copy “curveBrushTool.mll” to C:\Program Files\Autodesk\Maya8.5\bin\plug-ins；

Final load it in Maya.

Ø   3、HowToUse：

(1)If everything is ok , you will have the shelf  “YinHeTS” with the icon；

 

(2)Select the curves you want to paint , then click the icon above , finally paint the curve.

Red circle is the brush size , the Cross is the strength.

(3)With shift holding ，drag the mouse to change the brush size;

(4)With ctrl holding ，drag the mouse to change the brush strength;

(5)In the tool settings , check Lock Base on to lock the curve base , check it off to release its base , click Tool Help button for help；

Ø   4、Notes：

 This plugin is designed to brush Maya Hair curve .Enjoy it.
